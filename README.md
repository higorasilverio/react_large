# react-large [![NPM version][npm-image]][npm-url] [![Build Status][travis-image]][travis-url] [![Dependency Status][daviddm-image]][daviddm-url]

## Installation

```sh
$ npm install --save react-large
```

## Usage

```js
var reactLarge = require('react-large');

reactLarge('Rainbow');
```
